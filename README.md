- JSON
- JSON Schema

- [fastjson](https://github.com/alibaba/fastjson)

- [Jackson](https://github.com/FasterXML/jackson)
- [Klaxon](https://github.com/cbeust/klaxon)

- [Moshi](https://github.com/square/moshi)
